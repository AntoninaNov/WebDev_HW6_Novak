Endpoints

  • /nobels/ - Displays a list of Nobel Prize winners.
  • /nobels/<int:id>/ - Shows detailed information about a specific laureate.
  • /nobel/ - [POST] Create a new laureate entry (Use tools like Postman to interact with this endpoint).
  • /nobel/<int:id>/ - [DELETE] Remove a specific laureate from the list (Use tools like Postman).
